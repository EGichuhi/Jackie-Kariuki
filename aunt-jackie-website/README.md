# Aunt Jackie Website

## Overview
The Aunt Jackie website is designed to showcase the services and impact of Jackie Kariuki, a community leader dedicated to empowering individuals and organizations. This project includes a clean and modern design, with sections highlighting Jackie’s experience, services, and organizations she collaborates with.

## Project Structure
The project is organized as follows:

```
aunt-jackie-website
├── assets
│   ├── css          # Contains CSS files for styling the website
│   ├── js           # Contains JavaScript files for adding interactivity
│   └── img          # Contains image files used throughout the website
├── forms
│   └── contact-form.php  # Handles the submission of the contact form
├── index.html       # Main HTML file for the website
├── service-details.html  # Provides detailed information about services
└── README.md        # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd aunt-jackie-website
   ```

2. **Open the Project**
   Open the project in your preferred code editor.

3. **View the Website**
   Open `index.html` in a web browser to view the landing page.

4. **Contact Form**
   The contact form is located in `forms/contact-form.php`. Ensure your server is configured to handle PHP files to process form submissions.
